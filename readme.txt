Project consist of following:
1. BasePage class - page where we have methods and objects that will be used in all other page classes
2. GooglePage class - page where we appear during navigation to  google.com
3. GoogleSearchPage class - page with 10 results of searching
4. BaseTest class - class that consist before and after methods.
5. GoogleSearchTest class - test that consist of all input parameters and checks.
6. .gitignore file - file that consist information that shouldn't be pushed into github
7. pom.xml - file with all necessary libraries that imported and used in our test
8. README file - file with necessary information about following project

***How to run test**

1. Navigate to project -> src -> test -> java -> test -> GoogleSearchTest.
2. Click "Run test" button.
